## Quickstart Instructions

Really quick?

```
$ bin/quickstart.sh
```

More detailed:

```
$ bin/regen-certs
$ docker-compose down
$ docker-compose up -d
$ cd src/
$ composer install
$ npm install
$ cp .env.example .env
$ ./artisan key:generate
$ ./artisan jwt:secret -f
$ artisan clear-compiled
$ npm run dev
$ rm -rf database/database.sqlite
$ touch database/database.sqlite
$ ./artisan migrate:refresh --seed
$ ./artisan serve
```
